Panduan Lengkap Membaca Statistik Trading

Trading Style

🚀

Deskripsi

Mode strategi yang digunakan untuk trading.

Value: Scalping, Intraday, Swing

Status:

  • Fast (Scalping): Entry/exit dalam hitungan menit
  • Medium (Intraday): Hold beberapa jam
  • Slow (Swing): Hold harian/mingguan
📌 Cara Pakai:

Pilih mode sesuai profil risiko: Scalping untuk risiko tinggi, Swing untuk risiko rendah.

EMA Fast

📈

Deskripsi

Exponential Moving Average periode pendek.

Value: Angka harga (contoh: 3337.60)

Status:

  • Bullish → EMA Fast > EMA Slow
  • Bearish → EMA Fast < EMA Slow
📌 Cara Pakai:

Entry LONG jika status Bullish + harga di atas EMA Fast.

EMA Slow

📉

Deskripsi

Exponential Moving Average periode panjang.

Value: Angka harga (contoh: 3340.65)

Status:

  • Rising → Tren naik (bias bullish)
  • Falling → Tren turun (bias bearish)
📌 Cara Pakai:

Entry SHORT jika status Falling + harga di bawah EMA Slow.

RSI (P)

⚖️

Deskripsi

Relative Strength Index (mengukur momentum).

Value: 0-100 (contoh: 47.1)

Status:

  • Oversold (RSI < 30) → Potensi reversal naik
  • Overbought (RSI > 70) → Potensi reversal turun
  • Neutral → Pasar sideways
📌 Cara Pakai:

Entry LONG saat Oversold + konfirmasi indikator lain.

MACD Hist

📊

Deskripsi

Selisih antara garis MACD dan sinyalnya.

Value:
  • Positif (contoh: 0.4363) → Momentum naik
  • Negatif → Momentum turun

Status:

  • Bullish → Konfirmasi LONG
  • Bearish → Konfirmasi SHORT
📌 Cara Pakai:

Entry saat status berubah (misal: dari Bearish ke Bullish).

Volume

📦

Deskripsi

Volume transaksi.

Value: Angka volume (contoh: 1085)

Status:

  • High → Validasi sinyal kuat
  • Normal → Tidak ada insiden khusus
📌 Cara Pakai:

Abaikan sinyal jika volume Normal + ADX Weak Trend.

ADX

🌪️

Deskripsi

Average Directional Index (kekuatan tren).

Value: 0-100 (contoh: 17.1)

Status:

  • Strong Trend (ADX > 25) → Ikuti tren
  • Weak Trend (ADX < 25) → Pasar sideways
📌 Cara Pakai:

HOLD jika status Weak Trend + Confidence rendah.

Stochastic

🎯

Deskripsi

Indikator momentum (cepat/lambat).

Value: Dua angka (contoh: 66.1/72.1 = %K/%D)

Status:

  • Overbought (%K/%D > 80) → Potensi turun
  • Oversold (%K/%D < 20) → Potensi naik
  • Neutral → Tidak ada sinyal jelas
📌 Cara Pakai:

Entry SHORT saat status Overbought + garis %K memotong %D ke bawah.

AIR/ATR

🌊

Deskripsi

Average True Range (volatilitas).

Value: Angka volatilitas (contoh: 2.70)

Status:

  • High Vol → Pergerakan harga besar
  • Low Vol → Risiko false signal
📌 Cara Pakai:

Entry hanya jika status High Vol + Confidence ≥ B.

Confidence

🔐

Deskripsi

Skor kepercayaan sinyal (dihitung dari 8 faktor).

Value:
  • Persentase (contoh: 3%)
  • Grade: SSS (terbaik) sampai F (terburuk)

Status:

  • High (≥80% / Grade S↑) → Sinyal kuat
  • Medium (60-79% / Grade A-B) → Sinyal cukup
  • Low (<60% / Grade C-F) → JANGAN entry!
📌 Cara Pakai:

Entry hanya jika Grade ≥ B (minimal 60%).

Current Signal

🚦

Deskripsi

Rekomendasi entry saat ini.

Value:
  • LONG/SHORT (jika ada sinyal)
  • No Signal (tidak ada rekomendasi)

Status:

  • Bullish/Bearish → Konfirmasi arah
  • Neutral → Tunggu konfirmasi
📌 Cara Pakai:

Ikuti sinyal hanya jika Confidence ≥ 60%.

Strategi Trading Prioritas

📈 Strategi LONG

  • EMA Fast Bullish + EMA Slow Rising
  • RSI Oversold + MACD Bullish
  • Confidence ≥ B + Volume High
  • Stochastic Oversold + %K > %D
  • ADX > 25 (tren kuat)

📉 Strategi SHORT

  • EMA Fast Bearish + EMA Slow Falling
  • RSI Overbought + Stochastic Overbought
  • ADX Strong Trend + Confidence ≥ A
  • MACD Bearish + Volume naik
  • Harga di bawah EMA Slow

⚠️ Kondisi JANGAN ENTRY

  • Confidence = F/D
  • ADX = Weak Trend + Volume = Normal
  • EMA Fast/Slow status Neutral
  • Stochastic di tengah (40-60) + RSI netral
  • ATR Low Vol (volatilitas rendah)
  • Tampilan "No Signal"
📌 Penting:

Selalu konfirmasi minimal 3 indikator sejalan! Grade Confidence adalah filter utama.

Contoh Analisis Trading

Berikut contoh cara membaca tabel ScopeX untuk menentukan sinyal:

Indikator Value Status Analisis
EMA Slow 3350.00 Rising Tren utama naik (bullish)
RSI 28.5 Oversold Potensi reversal naik
MACD Hist 0.92 Bullish Momentum naik
ADX 32.1 Strong Trend Tren kuat
Confidence 85% A (High) Sinyal sangat valid
Signal LONG Bullish Rekomendasi: EXECUTE LONG!
📌 Kesimpulan:

✅ Tren kuat (ADX > 25) + ✅ Momentum naik (RSI oversold + MACD bullish) + ✅ Confidence tinggi (Grade A) → EXECUTE LONG!

Parameter Penting

⏱️ Delay Bars

Jeda minimal antar sinyal untuk menghindari false signal

Default: 1-5 bar

📏 EMA Dist %

Jarak minimal antara EMA Fast dan EMA Slow untuk validasi sinyal

Default: 0.03% - 0.08%

🕯️ Candle Dist %

Jarak minimal antara harga dan EMA Fast untuk validasi sinyal

Default: 0.02% - 0.07%

✅ Min Confidence

Batas minimal skor confidence untuk entry

Default: 60% (Grade B)
